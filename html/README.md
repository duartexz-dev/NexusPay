# 💰 MoneyFree

> Um sistema simples e eficiente para controle financeiro direto no navegador.

🔗 Acesse o projeto:  
https://duartexz-dev.github.io/MoneyFree/

---

## 📖 Sobre

O **MoneyFree** é uma aplicação web criada para ajudar no gerenciamento financeiro pessoal de forma prática.  
O sistema permite cadastrar informações do usuário, criar boletos e organizar renda e despesas, tudo sem precisar de login ou conexão com servidor.

---

## ✨ Funcionalidades

- Cadastro de usuário  
- Criação e gerenciamento de boletos  
- Organização de renda  
- Armazenamento automático dos dados  
- Interface responsiva  

---

## 🧠 Funcionamento

Todos os dados são armazenados utilizando o **LocalStorage**, o que permite que as informações permaneçam salvas no navegador mesmo após fechar ou atualizar a página.

---

## 🚀 Tecnologias

- HTML5  
- CSS3  
- Bootstrap  
- JavaScript  

---

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em praticar JavaScript e manipulação de dados no navegador, além de simular um sistema real de controle financeiro.

---

## ⚠️ Observações

- Os dados ficam salvos apenas no navegador  
- Não há sistema de login  
- Limpar o cache pode apagar as informações
